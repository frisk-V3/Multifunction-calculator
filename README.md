# PC Scientific Calculator  
PC 向けのシンプルで使いやすい科学電卓 Web アプリです。  
クリック操作とキーボード操作の両方に対応しています。

---

## 日本語 (Japanese)

### 概要
この電卓は PC での利用を前提に設計された Web ベースの科学電卓です。  
一般的な電卓に近いレイアウトで、四則演算や三角関数などの基本的な計算が可能です。

### 主な機能
- 四則演算  
- 括弧  
- 小数点  
- 三角関数（sin, cos, tan）  
- 平方根（√）  
- キーボード入力対応  
- Enter キーで計算  
- Backspace で削除  
- Escape でクリア  

### 使い方
1. ブラウザで `index.html` を開きます。  
2. ボタンをクリックするか、キーボードで入力します。  
3. Enter キーで計算結果が表示されます。

## English

### Overview
This is a web-based scientific calculator designed for PC use.  
It provides a familiar calculator layout and supports basic scientific functions.

### Features
- Basic arithmetic operations  
- Parentheses  
- Decimal input  
- Trigonometric functions (sin, cos, tan)  
- Square root (√)  
- Full keyboard input support  
- Enter key to calculate  
- Backspace to delete  
- Escape key to clear  

### How to Use
1. Open `index.html` in your browser.  
2. Use the on-screen buttons or your keyboard.  
3. Press Enter to calculate the expression.

## 한국어 (Korean)

### 개요
이 계산기는 PC 사용을 위해 설계된 웹 기반 과학용 계산기입니다.  
일반적인 계산기와 비슷한 배열을 가지고 있으며 기본적인 과학 계산 기능을 제공합니다.

### 주요 기능
- 사칙연산  
- 괄호  
- 소수 입력  
- 삼각함수 (sin, cos, tan)  
- 제곱근 (√)  
- 키보드 입력 지원  
- Enter 키로 계산  
- Backspace 로 삭제  
- Escape 로 초기화  

### 사용 방법
1. 브라우저에서 `index.html` 파일을 엽니다.  
2. 화면의 버튼 또는 키보드를 사용합니다.  
3. Enter 키를 눌러 계산합니다.
